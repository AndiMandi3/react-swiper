# Тестовое задание на React
## Требования для запуска
- Node.JS: 22.14.0

## Установка зависимостей
После импорта проекта, прописать в терминале в текущей директории: ```npm install```

## Запуск
- Для проверки приложения линтером: ```npm run lint```
- Для запуска в dev-режиме: ```npm run dev```
- Для сборки: ```npm run build```
- Для просмотра собранного проекта: ```npm run preview```
